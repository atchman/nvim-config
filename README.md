# my neovim config

# Requirements
- neovim => 0.10.3
- git => 2.47.1
- gcc =>
- ripgrep => 14.1.1
- fd => 10.2.0
- fzf
- wl-copy

# Build && Install

```shell
git clone https://github.com/neovim/neovim
cd neovim && make CMAKE_BUILD_TYPE=RelWithDebInfo
sudo make install
```
